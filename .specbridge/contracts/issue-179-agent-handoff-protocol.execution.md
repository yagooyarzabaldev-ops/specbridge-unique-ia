## Goal

Give life to the agent input/output artifact pointers declared in orchestration manifests. Add a specbridge-handoff command that records an agent's output artifact, marks the agent completed in the orchestration manifest, activates the next agent in the chain, and transitions orchestration status planned -> in_progress -> completed. Extend validate-orchestrations.ps1 to verify handoff consistency: completed agents must have their output artifact on disk, agent progression must be sequential, and orchestration status must match agent states. Add CLI tests and keep smoke green.
