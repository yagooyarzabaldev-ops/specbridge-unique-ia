## Purpose

Define a controlled Claude Code execution task.

## Inputs

- goal
- allowed scope
- blocked scope
- acceptance criteria
- required validations
- stop conditions

## Allowed Execution

Only low-risk tasks may use this template initially.

## Blocked Execution

- secrets
- production configuration
- billing
- destructive operations
- deployment automation
- product runtime implementation without explicit policy

## Final Report

Execution must produce an evidence-based final report.
